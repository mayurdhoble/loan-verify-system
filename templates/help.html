üîç Frequently Asked Questions

Q: What documents do I need to upload?

You need exactly 3 PDF documents: DFR (Dealer Financing Request), Purchase Agreement, and Title Document. All three are required for verification.

Q: What is a member ID?

A member ID is a unique number (1, 2, 3...) automatically assigned to each uploaded profile. Use this ID to search and verify members.

Q: What if my documents don't have all the required fields?

The system will extract whatever information is available. Missing fields will be marked as "N/A" and will affect the similarity score.

Q: How is the similarity score calculated?

The system compares 5 fields: applicant name, VIN, loan amount, routing number, and account number. The score is the average match percentage across all fields. 80% or higher = Verified.

Q: What does "Verified" vs "Not Verified" mean?

Verified: 80%+ similarity between documents and web profile (high confidence match)
Not Verified: Less than 80% similarity (manual review recommended)

Q: Can I re-verify a member?

Yes! Search for the member ID and click "Start Verification" again. A new verification record will be created.

Q: How do I export verification results?

Go to History page ‚Üí Click "Export CSV" button at the top right. This downloads all verification records as a CSV file.

Q: What file formats are supported?

Only PDF files are supported for document uploads. The system uses OCR to extract text from images within PDFs.

Q: Why is my verification taking long?

The verification process involves OCR scanning, AI extraction, and web scraping. This typically takes 10-30 seconds depending on document size and complexity.

Q: Can I delete a member or verification?

Currently, the system doesn't have a delete feature in the UI. Records can be deleted directly from the database using SQL commands.

Q: What if the web profile URL is not accessible?

If the URL returns an error or is blocked, Agent 2 will fail to extract data. Ensure the URL is publicly accessible.

Q: How accurate is the OCR extraction?

OCR accuracy depends on document quality. For best results:

  • Use high-resolution scans (300+ DPI)
  • Ensure text is clear and not blurry
  • Avoid skewed or rotated documents

üîß Troubleshooting

Issue: "No text extracted from document"

Solution: The PDF may be image-based without text. Ensure documents contain selectable text or high-quality images for OCR.

Issue: "Web extraction failed"

Solution: Check if the URL is correct and accessible. Some websites may block automated scraping.

Issue: Low similarity score despite correct data

Solution: Check the "Mismatched Fields" details. Small differences in formatting (e.g., "John Smith" vs "JOHN SMITH") might cause mismatches.
